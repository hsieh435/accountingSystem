# Project

一個記帳系統，支援交易紀錄、報表分析、登入驗證等功能。


## 功能特色

- 使用者登入 / JWT 驗證
- 收入支出管理
- PostgreSQL 整合

## 安裝方式

```bash
git clone https://github.com/hsieh435/accountingSystem.git
cd accountingSystem
npm i
npm run dev


前端使用技術
UI / UX：SCSS + Tailwind + Nuxt UI
框架：Nuxt 3
資料視覺化：plotly.js


後端使用技術
node.js

資料庫
pgadmin
